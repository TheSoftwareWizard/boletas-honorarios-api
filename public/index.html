API de Boletas de Honorarios

API REST para datos de retención de boletas de honorarios profesionales chilenos

Descripción General

Esta API proporciona datos de retención para boletas de honorarios profesionales chilenos, incluyendo tasas actuales e históricas por año.

Endpoints de la API

GET /v1/retention

Retorna la tasa de retención actual y metadatos relacionados.

Respuesta:


                

            


            

                

                    GET
                    /v1/retention/history
                

                
Retorna las tasas de retención históricas por año.

                
                

                    
                    
                

                
                

                    
Respuesta:
